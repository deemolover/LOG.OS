# Plotting

`matplotlib` 是一个可用于简单的数值分析绘制的库。

```python
import matplotlib.pyplot as plt
plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.xlabel('some variables')
plt.ylabel('some numbers')
plt.show()
```
